# Energy-Economy Nexus: Analyzing the Co-Movement of U.S. Electricity Price Shocks and Global Macroeconomic Indicators During the 2022–2024 Volatility Period
## Project Overview
This project investigates the relationship between U.S. electricity price fluctuations and global macroeconomic indicators during the 2022–2024 period, characterized by significant market volatility. The project integrates multiple economic datasets through an end-to-end ETL pipeline and data warehouse architecture to support multidimensional analysis.

The study explores how electricity price shocks interact with key macroeconomic variables, providing insights into energy market dynamics and their broader economic implications.
## Objectives
Build an integrated data warehouse for energy and macroeconomic datasets.
Design and implement ETL workflows for heterogeneous data sources.
Analyze the co-movement between electricity prices and macroeconomic indicators.
Perform multidimensional analysis using OLAP techniques.
Visualize economic trends through interactive dashboards.
